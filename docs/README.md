# trasta とらすた サイト仕様ドキュメント

このディレクトリは、`trasta` / `とらすた` の個人サイトを実装するための仕様置き場です。

## 採用方針

- フレームワークは **TanStack Start** を採用します。
- デプロイ先は **Cloudflare Workers** を前提にします。
- 開発体験として **Vite+** を導入します。
- サイトの主役はブログです。`about` と `works` はブログを補助する導線として扱います。
- デザインは白黒ベースで、添付アイコン由来の黄・ピンク・ミント・ラベンダーなどを小さく使います。

## ドキュメント

- [Product Spec](./product-spec.md): サイト全体の目的、ページ、機能要件。
- [Technical Spec](./technical-spec.md): 技術スタック、ルーティング、データ、デプロイ、admin案。
- [Design Spec](./design-spec.md): デザイン方針、アニメーション、画像素材の扱い。

## 重要な表記ルール

- 正しい表記は `trasta` / `とらすた` / `trasta.dev` です。
- `TRASTA`、`トラスタ`、その他の大文字表記・カタカナ表記は使いません。
